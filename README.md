# Python Console Typing App

Pythonで作成したコンソール上で動作するタイピングアプリ

## 導入

---

以下コマンドを実行 : 

```bash
git clone https://github.com/koukemo/PythonConsoleTypingApp.git
```

## 実行手順

---

```bash
cd PythonConsoleApp
python typing_app/main.py
```
